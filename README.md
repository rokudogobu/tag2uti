
# tag2uti

Simple C program to get UTI (= Uniform Type Identifier) from filename extension or MIME type on Mac.

## Usage

    usage: tag2uti <ext | mime>...

## Example

    $ tag2uti c text/html bash
    public.c-source
    public.html
    public.bash-script com.apple.xcode.bash-script
